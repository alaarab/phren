Return Format #

All MCP tools return structured JSON with a consistent shape:

{
  "ok": true,
  "message": "Found 3 results",
  "data": { ... }
}

// On error:
{
  "ok": false,
  "error": "Project not found: foo"
}

Maintenance tools (phren config, phren maintain) are CLI-only and not exposed via MCP.
