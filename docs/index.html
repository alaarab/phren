Why cortex

Built around how
Claude actually works.

Context windows are finite and tokens cost money. Cortex makes Claude precise about what it loads, pulling only the relevant slice instead of the whole file.

Get started

Up in three commands.

No config files. No accounts. Add the MCP, install the plugin, and you're done.

1
Add the MCP server
Registers cortex with Claude Code and gives it the search tools.
$ claude mcp add cortex -- npx @alaarab/cortex
2
Install from the plugin marketplace
Adds the skills package: /cortex:learn, /cortex:sync, and more.
$ /plugin marketplace add alaarab/cortex
3
Install the cortex plugin
Activates the plugin. Run this inside a Claude Code session.
$ /plugin install cortex@cortex
Skills

Skills for the
whole workflow.

Run any skill inside a Claude Code session. They're markdown files in your cortex, so you can read them, tweak them, or swap them out entirely.

/cortex:sync
Pull skills to the current project or push changes back to your cortex repo.
/cortex:learn
Capture what this session figured out. Commits to your knowledge base, pushes to remote.
/cortex:init
Initialize a new cortex knowledge base at ~/.cortex and wire up the git remote.
/cortex:discover
Scan a project and auto-generate a structured summary for your knowledge base.
/cortex:consolidate
Merge fragmented notes into a clean, single-document summary for a project.
/humanize
Kill the AI voice. Checks word lists, structural patterns, and code tells in one pass.
/swarm
Split work across 2–5 agents running in parallel. Frontend, backend, tests: they coordinate through a shared task list and report back when done.
/backlog
Persistent project task queue. Add items, prioritize, or swarm the backlog with team agents.
/pipeline
Git status, recent commits, test results, open PRs. A quick read of where things actually stand.
/release
Ship a version. Version bump, changelog, tag, publish to npm or PyPI.
/creative
Design pass that asks what a good designer would: spacing, typography hierarchy, does it look made or filled in from a template.
