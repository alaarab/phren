# templates/

Generic scaffolding templates and examples used by the `cortex-init` skill when no `--template` flag is provided.

Contains a bare `project/` skeleton (CLAUDE.md, summary.md, backlog.md with placeholder fields), a `MEMORY-project.md` template for auto-memory files, and `examples/` with filled-in summary.md samples showing what good project summaries look like.

This directory is distinct from `starter/templates/` which holds typed templates (frontend, library, etc.) for the CLI `--template` flag. This one provides the default scaffolding and reference examples for the skill-based init flow.
