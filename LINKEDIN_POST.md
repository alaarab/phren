## the post

i was building a framework for generating audio plugins on demand. claude didn't think it was possible. so i taught it. then the chat ended and i had to teach it again. and again.

every new chat i was burning tokens re-explaining my own project. eventually got annoyed enough to build phren.

phren is a memory layer for ai agents. one command to install:

`npx @phren/cli init`

point it at a repo and boom, claude remembers. so does copilot, cursor, codex. same memory, all four. ships as a cli and a vscode extension with a fragment graph and session timeline.

your data lives as markdown in your own git repo. imagine that, actually owning your data in 2026. lol.

open source, mit. on npm as `@phren/cli`. vscode extension is `phren` on the marketplace.

try it. break it. tell me.
